SYSTEM:

- Output exactly one JSON object matching the Instruction schema: {id, description, template, difficulty, time_limit, success_criteria[]}.
- Use template = "desktop".
- success_criteria must be concrete and machine-testable (no vague goals). Prefer these predicates:

- Do not include any keys other than required schema keys.

- If the task mentions opening or checking an app/page, use element_text_contains on a distinctive heading or label (e.g., Settings, Files, Browser).
- If the task mentions creating or finding a file, use file_exists:/home/user/... and/or element_text_contains for confirmation text.
- Derive a short id by slugifying the description (lowercase, dashes).
